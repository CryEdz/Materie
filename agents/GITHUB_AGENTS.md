Uso consigliato:
- laboratori AWS, Azure, GCP
- Docker, Kubernetes, Terraform
- setup, deploy, troubleshooting, cleanup

## Coding Coach ITS
Uso consigliato:
- esercizi Python, C#, C/C++
- REST API, algoritmi, debug guidato
- spiegazioni tecniche con esempi

## Exam Prep ITS
Uso consigliato:
- piani di studio settimanali
- simulazioni prova e quiz
- ripasso per materia con priorita

Per richiamarli, seleziona l'agente in chat oppure usa prompt dedicati in `.github/prompts`.
