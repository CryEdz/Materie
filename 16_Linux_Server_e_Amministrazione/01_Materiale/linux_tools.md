# 🐧 Linux Server Tools
> Guida di riferimento rapido per Debian / Ubuntu

---

## 🖥️ Monitoraggio Sistema

| Tool | Descrizione |
|------|-------------|
| `htop` | Monitor processi interattivo (meglio di `top`) |
| `btop` | Monitor risorse moderno e visuale |
| `glances` | Overview completo del sistema |
| `nmon` | Monitor performance per server |
| `vmstat` | Statistiche memoria, CPU, I/O |
| `iostat` | Statistiche I/O dischi |
| `sar` | Raccolta dati storici di sistema |
