# Git Tools

Una guida essenziale e strutturata ai comandi e strumenti più utilizzati per il versionamento del codice con Git.

## Contenuto

Il repository contiene:

- **`Tool.md`** — Elenco completo di comandi Git organizzati per categoria:
  - Configurazione
  - Repository (init, clone)
  - Modifiche e Commit
  - Branch e Merge
  - Remote e Sincronizzazione
  - Ispezione e Log
  - Stash
  - Reset e Revert
  - Tag
  - Rebase e Cherry-Pick
  - Submodule
  - Bisect e Debug
  - Gitignore e Attributi
  - Hook
  - Worktree
  - Strumenti Ausiliari (gh, tig, lazygit, git-lfs, git-flow, pre-commit)

Ogni comando include: nome, descrizione, sintassi, esempio d'uso e note.

## Struttura del file

```markdown
### `nome-comando`
**Descrizione breve.**

- **Comando:** `comando principale`
- **Esempio:** `esempio d'uso`
- **Note:** Note aggiuntive
```

## Requisiti

- **Git:** Installazione di Git (https://git-scm.com)
- **Strumenti ausiliari:** Opzionali, installabili singolarmente (gh, tig, lazygit, ecc.)
