# Data Analytics & Visualization Tools

Una guida essenziale e strutturata ai tool e comandi più utilizzati per analisi dati, visualizzazione e data science.

## Contenuto

Il repository contiene:

- **`Tool.md`** — Elenco completo di strumenti organizzati per categoria:
  - Librerie Python per Analisi Dati (pandas, numpy, polars)
  - Librerie Python per Visualizzazione (matplotlib, seaborn, plotly, bokeh)
  - Notebook e Ambienti Interattivi (Jupyter, JupyterLab, Colab)
  - Machine Learning (scikit-learn, XGBoost, TensorFlow, PyTorch)
  - Business Intelligence (Power BI, Tableau, Looker, Metabase, Grafana)
  - Database Analitici (ClickHouse, DuckDB, Druid, InfluxDB)
  - ETL e Data Pipeline (Spark, dbt, Airflow, Prefect, Kafka)
  - Time Series e Streaming (Prophet, Streamlit, Dash)
  - Statistical Computing (R, ggplot2, dplyr)
  - Strumenti Ausiliari (csvkit, ydata-profiling, great_expectations, mlflow)

Ogni tool include: nome, descrizione, comando principale, esempio d'uso e note.

## Struttura del file

```markdown
### `nome-tool`
**Descrizione breve.**

- **Comando:** `comando principale`
- **Esempio:** `esempio d'uso`
- **Note:** Note aggiuntive
```

## Requisiti

- **Python:** 3.8+ con pacchetti installabili via pip
- **R:** Opzionale, per sezioni statistiche
- **BI Tools:** Power BI Desktop (gratuito), Tableau Public (gratuito)
- **Database:** ClickHouse, DuckDB (a seconda delle esigenze)
