# Prompt — Docker compose

Contesto: `17_Container_Docker_Kubernetes/labs/17-Docker_compose`

Obiettivo: scrivi un `docker-compose.yml` che avvii un'app web (porta 8080) e un database Postgres con volumi persistenti. Fornisci i comandi per build, avvio e controllo dei log.

Richiedo: `docker-compose.yml`, eventuali `Dockerfile` necessari e i comandi `docker compose up --build -d`, `docker compose logs -f`.
